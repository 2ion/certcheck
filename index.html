Certcheck

CLI tool to check x.509 cert validity on a set of TLS/SSL servers

Basic tool to check a bulk of host:port pairs if the SSL certificate used by the SSL/TLS server on that port is valid. The author uses it to generate email reports of the status of his SSL properties.

The routine is SNI-aware. STARTTLS support is going to be implemented sometime later.

Synopsis

usage: certcheck [-h] [-l FILE] [-t N] [host:port [host:port ...]]

Batch x509 certificate validity checker

positional arguments:
  host:port             TLS/SSL host

optional arguments:
  -h, --help            show this help message and exit
  -l FILE, --host-list FILE
                        File with a host:port list
  -t N, --threads N     Number of servers to check in parallel

Output sample

$ ./certcheck blog.fefe.de heise.de hackerne.ws
